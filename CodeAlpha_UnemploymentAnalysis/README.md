# Unemployment Analysis with Python

## Project Overview
This project analyzes unemployment rate data using Python to identify trends, patterns, and the impact of Covid-19 on employment levels.

## Objective
To perform data analysis and visualization on unemployment data and derive meaningful insights.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
Unemployment Rate Dataset

## Analysis Performed
- Data Cleaning
- Missing Value Analysis
- Unemployment Trend Analysis
- State-wise Unemployment Analysis
- Covid-19 Impact Analysis

## Visualizations
- Unemployment Trend Graph
- State-wise Analysis Chart
- Covid-19 Impact Graph

## Key Insights
- Identified unemployment trends over time.
- Compared unemployment rates across different states.
- Analyzed the effect of Covid-19 on unemployment levels.
