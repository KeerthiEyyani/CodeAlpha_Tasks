# Iris Flower Classification using Machine Learning

## Overview
This project classifies Iris flowers into three species:
- Iris Setosa
- Iris Versicolor
- Iris Virginica

using Machine Learning techniques.

## Objective
To build a classification model that accurately predicts the species of an Iris flower based on its sepal and petal measurements.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Algorithm
Random Forest Classifier

## Dataset
Iris Dataset from Scikit-learn

## Results
The model achieved high accuracy in predicting flower species and was evaluated using:
- Accuracy Score
- Classification Report
- Confusion Matrix
