# Car Price Prediction using Machine Learning

## Overview
This project predicts car selling prices based on factors such as year, present price, kilometers driven, and fuel type.

## Objective
To build a machine learning regression model that can estimate the selling price of a car.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Algorithm
Random Forest Regressor

## Features Used
- Year
- Present Price
- Kilometers Driven
- Fuel Type

## Evaluation Metrics
- R² Score
- Mean Absolute Error (MAE)

## Outputs
- Dataset Preview
- Correlation Heatmap
- Actual vs Predicted Graph
- Model Performance Chart
